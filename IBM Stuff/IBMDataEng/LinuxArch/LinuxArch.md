### Filesystem
- "/" root
	- "/bin"
	- "/usr"
		- user programs
	- "/home" or "~"
		- personal files
	- "/boot"
		- system boot files
	- "/media"
		- temp media (CD, USB)

## Commands

### Info 
- `whoami` - suername
- `id` - user id, group id
- `uname` - operating system name (neofetch?)
- `ps` - processes
- `top` -resource  usage
- `df` - mounted file systems
- `man` - manual for anything

### networking
- `hostname` - prints hostname
- `ifconfig` - display sys net interfaces
